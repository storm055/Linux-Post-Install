# 🐧 Linux Post Install

A simple and practical post-install guide for Linux systems.

This repository helps you quickly set up a fresh Linux installation with essential tools, system updates, drivers, and development utilities.

It supports:

- Arch Linux
- Debian
- Ubuntu
- and other derivatives

---

## 🚀 What this includes

- System updates
- Essential packages
- Development tools
- Git + SSH setup
- Firewall configuration
- Flatpak setup
- Cleanup commands
- Optional improvements

---

## 💬 Why this exists

A fresh Linux install is powerful, but not ready out of the box.

This guide helps you turn it into a usable development environment in minutes.
