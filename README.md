     Linux Post Install Script

A simple and practical post-install guide for Linux systems.

This project helps you turn a fresh Linux installation into a fully usable development environment in minutes.

Works on:

- Arch Linux
- Debian
- Ubuntu
- and other Linux-based distributions

---

## ⚡ Quick Start

### 1. Clone the repo

```bash
git clone https://github.com/your-username/linux-post-install.git
cd linux-post-install
```

---

### 2. Make script executable

```bash
chmod +x post-install.sh
```

---

### 3. Run it

```bash
./post-install.sh
```

---

## 🚀 One-line install (advanced)

If you trust the script, you can run it directly:

```bash
curl -fsSL https://raw.githubusercontent.com/your-username/linux-post-install/main/post-install.sh | bash
```

---

## 💻 Why this exists

A fresh Linux install is powerful, but not ready out of the box.

This project exists to remove repetitive setup steps and give you a clean, ready-to-use development environment.

Think of it as a **base system bootstrap**.

---

## 🔥 Dotfiles / Power user vibe

This repo can be used as:

- A post-install bootstrap script
- A base setup for new machines
- A starting point for your own dotfiles
- A minimal Linux dev environment initializer
