# 🤝 Contributing to Linux Post Install

First of all, thank you for considering contributing to this project! 🚀

This project is a simple post-install script for Linux systems, and any help to improve it is welcome.

---

## 🐧 What this project is about

This repository helps users quickly set up a fresh Linux installation with essential tools, updates, and configurations.

Supported systems:
- Arch Linux
- Debian
- Ubuntu
- Other Linux-based distributions

---

## 💡 How you can contribute

You can help by:

- 🐛 Reporting bugs
- ✨ Suggesting improvements
- ⚙️ Improving the `post-install.sh` script
- 📦 Adding support for more Linux distributions
- 📝 Improving documentation (README, guides, etc.)

---

## 🔧 Getting started

1. Fork the repository
2. Clone your fork

```bash
git clone https://github.com/your-username/linux-post-install.git
```

3. Create a new branch

```bash
git checkout -b my-feature
```

4. Make your changes

5. Commit your changes

```bash
git commit -m "Add: short description of your change"
```

6. Push to your branch

```bash
git push origin my-feature
```

7. Open a Pull Request

---

## 📌 Guidelines

- Keep changes simple and focused
- Test your script changes before submitting
- Follow the existing style of the project
- Keep it beginner-friendly

---

## ⚠️ Important

This is a lightweight utility project.  
Please avoid adding unnecessary complexity or features that break simplicity.

Feel free to open an issue if you have questions or suggestions.
